  type: feedback
---

훅이 "ctx X%" 또는 "새 세션 권장" 메시지를 보내도, 실제로 작업을 계속할 수 있는지는 AI 스스로 판단하라.

**Why:** 훅의 ctx% 계산이 부정확할 수 있고, 사용자는 실제 컨텍스트 상태를 더 잘 안다. 훅 경고를 그대로 전달하며 작업을 중단하면 불필요한 세션 전환을 강요하게 된다.

**How to apply:**
- 훅 ctx% 경고 → 참고 메모만 할 것, 작업 중단 판단은 AI가 직접 내림
- 실제로 응답이 불가능하거나 컨텍스트 오류가 발생할 때만 사용자에게 세션 전환 제안
- "훅이 xxx라고 했으니" 식의 맹목적 전달 금지
